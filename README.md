## Template Vue

## Stack

Vue3,
Tailwind,
I18n,
Pinia,
Архитектура - FSD

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

### Start Tailwind

```sh
npx tailwindcss -i ./src/app/styles/input.css -o ./src/app/styles/styles.css --watch

```
